指令 動作 備註
hexo new “PostName” 發布新文章
hexo clean 清空靜態檔緩存
hexo server Run server Server
hexo generate Generate static files 產生靜態檔，Generating
hexo deploy Deploy to remote sites
hexo build 編譯
hexo init 初始化Hexo資料夾
hexo -v 確認版本
